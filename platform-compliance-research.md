# 中国主流社交平台内容合规调研报告

> 调研范围：B站（哔哩哔哩）、抖音、小红书、知乎、微信视频号/公众号  
> 重点维度：平台规则、推流机制、用户画像、内容红线（禁止内容/限流/封号）、相关法律法规

---

## 二、各平台推流机制

### 2.1 B站（哔哩哔哩）

**推荐算法核心因素：**
- **互动率（完播率、点赞、投币、收藏、弹幕量）** 权重极高
- **关注者点击率**（Follower Click Rate）：发布后粉丝打开比例
- **内容标签与用户兴趣匹配**
- **新人扶植**：新账号前3-5个视频有额外流量扶植
- **系列内容加权**：连载/合集内容权重更高

**推流阶段：**
1. 冷启动：先投少量流量测试数据
2. 社区反馈好 → 进入「主站推荐」
3. 优质内容 → 进入「热门榜」
4. 长期优质 → 进入「每周必看」等官方栏目

### 2.2 抖音

**推荐算法核心因素：**
- **完播率**（最重要指标，优于点赞率）
- **互动率**（点赞、评论、收藏、转发）
- **账号权重**（注册时长、发稿量、违规记录）
- **内容标签+用户兴趣匹配**
- **同城定位**（对本地生活类内容）
- **DOU+加热**：付费推广可直接影响推荐量

**推流阶段：**
1. 200-500基础播放（冷启动）
2. 数据达标 → 第二级流量池（3000-5000）
3. 逐级递增，优质可达500万+播放
4. 登上榜单 → 进入热搜池

### 2.3 小红书

**推荐算法核心因素：**
- **笔记互动率**（收藏、点赞、评论）
- **封面图点击率**（CTR）——小红书极度重视
- **账号权重**（违规会影响笔记曝光）
- **关键词布局**：搜索流量占比较高
- **真实用户 vs 机器人识别**：平台严厉打击刷量

**推流阶段：**
1. 先推给部分关注者/标签用户测试
2. 数据好 → 进入「发现页」推荐
3. 爆款笔记 → 进入「搜索关键词排名前列」

### 2.4 知乎

**推荐算法核心因素：**
- **回答质量**（赞同、反对、没有帮助）
- **专业性标识**（蓝标认证、专业徽章）
- **内容字数**：深度长文权重更高
- **关注者粘性**：垂直领域深耕权重高
- **搜索流量**：知乎百度权重高，SEO效果好

**推流阶段：**
1. 先推送给关注该问题/话题的用户
2. 高赞回答 → 进入「知乎日报/精选」
3. 优质回答 → 进入「搜索结果高权重位置」

### 2.5 微信视频号

**推荐算法核心因素：**
- **微信社交关系链**：朋友圈点赞、群分享是核心传播路径
- **完播率**
- **互动率**（点赞、收藏、分享）
- **公众号绑定**：视频号与公众号互通
- **地理位置**：同城内容优先展示
- **热度加权**：当前话题热度

**推流阶段：**
1. 先推给微信好友/群
2. 好友互动好 → 进入「朋友赞过」内容流
3. 扩量后 → 进入「推荐」信息流

### 2.6 微信公众号

**推荐算法核心因素：**
- **订阅用户基础打开率**（公众号push打开率）
- **在看/点赞数**（归入「朋友再看」）
- **留言互动率**
- **打赏率**（对原创内容的正向反馈）
- **转发率**：订阅用户转发朋友圈带来新增关注

**推流特点：**
- 无平台算法推荐，主要靠粉丝主动触达
- 「看一看」算法会推荐好友在读/在看的文章
- 2023年后微信尝试在「看一看」引入更多算法推荐

## 三、内容红线（禁止内容/限流/封号标准）

### 3.1 B站内容红线

**绝对禁止内容（上传即删+封号）：**
- 违反法律法规内容（分裂国家、暴恐、邪教等）
- 恶意诋毁国家领导人/英烈
- 色情低俗内容（含软色情、ACG色情擦边）
- 政治敏感内容（未经认证的国际政治分析）
- 未经授权的影视综艺剪辑（版权红线）
- 血腥暴力内容（含恐怖游戏过度展示）
- 非法集资/诈骗内容
- 个人信息泄露（如人肉搜索）
- 境外VPN推广内容

**常见限流原因：**
- 标题党/封面党（过度封面与内容不符）
- 刷互动量（机器人点赞/投币）
- 频繁发低质量水视频
- 搬运内容未标注「自制」
- 商业广告未走「花火平台」
- 引战拉踩内容

**封号标准（阶梯处罚）：**
- 小黑屋（限时禁止发言）：评论违规
- 冻结账号（1天-30天）：轻度违规
- 永久封禁：严重违规或累犯
- 风纪委员机制：用户举报→风纪委员投票→官方执行

### 3.2 抖音内容红线

**绝对禁止内容（永久封禁风险）：**
- 违反「四个意识」「两个维护」等政治要求内容
- 涉政敏感话题（领导人名字/绰号/职务称呼混用等）
- 涉军乱局：军事装备非授权拍摄、军人影像
- 暴恐血腥内容
- 未成年人参与直播/打赏
- 色情低俗：任何性暗示内容
- 赌博/彩票推广
- 非法药品（迷药等）
- 伪造政府公文/证件
- 挑战平台底线内容（如恶意炒CP、炒热度）

**常见限流原因：**
- 搬运/抄袭他人内容（无原创标识）
- 隐形营销（软广告未标注）
- 诱导点赞/关注（「给我点赞就发你...」）
- 内容过短（低于3秒无效播放）
- 过度特效/贴纸影响观看
- 涉及争议话题引战
- 直播中提及其他平台（不允许引流）

**封号标准：**
- A级违规：警告+限流
- B级违规：短期封禁（1-7天）
- C级违规：30天封禁+扣除信用分
- D级违规：永久封禁
- 信用分12分制，满分12分，扣完封禁

### 3.3 小红书内容红线

**绝对禁止内容：**
- 涉政内容（国家领导人、领土主权等）
- 虚假宣传/违法广告（涉及医疗/金融/保健品）
- 色情低俗内容
- 未成年人内容（未成年人直播/短视频）
- 炫富/拜金内容（平台近年大力整治）
- 煽动对立（性别对立、地域歧视）
- 虚假信息/谣言（尤其医疗/食品安全）
- 侵权内容（盗图、抄袭笔记）
- 违禁药物相关内容
- 展示封建迷信内容

**常见限流原因：**
- 敏感词触发（笔记中含违禁词被降权）
- 大量重复发布相似内容（「水笔记」）
- 私信引流到其他平台
- 虚假种草（花钱写笔记的软广告）
- 封面/图片质量过低
- 长期不活跃后突然活跃

**封号标准：**
- 笔记隐藏：单篇内容删除
- 账号禁言：1-30天禁止发言
- 永久封禁：严重违规
- 「薯条」投放限制：违规账号不可购买推广

### 3.4 知乎内容红线

**绝对禁止内容：**
- 违反法律法规（参考下方法律章节）
- 政治敏感内容
- 恶意诋毁他人/造谣
- 侵权内容（抄袭、未授权转载）
- 垃圾广告（硬广、水军内容）
- 色情低俗内容
- 非法信息交易（证件、假货等）

**常见限流原因：**
- 「答非所问」：回答与问题关联性低
- 反对/没有帮助积累过多
- 争议性引战内容
- 匿名造谣（匿名机制已调整）
- 过度营销性质内容

**封号标准：**
- 折叠回答：内容被折叠但账号正常
- 禁言7天/30天
- 永久封禁
- 专业身份认证被撤销

### 3.5 微信视频号/公众号红线

**微信视频号绝对禁止内容：**
- 违反微信公众平台运营规范的一切内容
- 涉政内容（视频号审核比公众号更严）
- 未经授权的品牌/商标展示（易被投诉）
- 诱导分享（「转发送礼」类内容）
- 刷量/买粉行为
- 境外社交平台内容引流
- 低俗色情内容
- 虚假夸大广告（涉及医疗/金融）

**微信公众号绝对禁止内容：**
- 恶意营销/欺诈内容
- 未取得互联网新闻信息服务许可的新闻类内容
- 违规转载/抄袭原创内容
- 违法广告（尤其是医疗、金融、P2P）
- 政治类内容（需特别许可）
- 推送内容与注册主体资质不符

**封号/处罚标准：**
- 警告+内容删除
- 限制功能（留言/打赏）7-30天
- 封禁账号（可申诉）
- 恶意营销累犯直接封禁
- 视频号与公众号联动违规会双向处罚

## 四、相关法律法规汇总

### 4.1 法律层级分类

| 层级 | 名称 | 施行时间 | 重点内容 |
|------|------|---------|----------|
| **法律** | 《网络安全法》 | 2017.6.1 | 数据安全、个人信息保护、网络运营者责任 |
| **法律** | 《著作权法》 | 2021.6.1（修订） | 作品定义、转载规则、侵权赔偿提升 |
| **法律** | 《广告法》 | 2015.9.1（修订） | 虚假广告绝对化用语、医疗金融广告限制 |
| **行政法规** | 《互联网信息服务管理办法》 | 2011.1.8（修订） | ICP许可、内容禁止规定 |
| **行政法规** | 《网络信息内容生态治理规定》 | 2020.3.1 | 违法信息、不良信息、正能量信息分类 |
| **部门规章** | 《互联网新闻信息服务管理规定》 | 2022.5.1 | 新闻转载需许可资质 |
| **部门规章** | 《互联网广告管理办法》 | 2023.5.1 | 种草广告需标注「广告」、软文规则 |

### 4.2 重点法条解析

#### 《网络安全法》重点条款

| 条款 | 内容要点 | 平台合规要求 |
|------|---------|------------|
| 第24条 | 用户真实身份信息登记 | 平台需落实实名制 |
| 第41条 | 个人信息收集「最小必要」原则 | 不得过度收集用户数据 |
| 第44条 | 不得非法获取或出售个人信息 | 违规者最高罚款100万元 |
| 第47条 | 平台对违法内容有「发现后立即停止传输」义务 | 平台须有内容审核团队 |
| 第50条 | 监管部门有权要求平台删除/屏蔽内容 | 政府管控工单下发机制 |

#### 《著作权法》2021修订重点

| 条款 | 内容要点 | 对内容创作者影响 |
|------|---------|---------------|
| 第3条 | 作品定义扩展至「短视频」「直播内容」 | B站/抖音视频属于作品 |
| 第52条 | 侵权赔偿上限提升至500万元 | 盗版搬运风险极高 |
| 第54条 | 平台「避风港原则」适用条件收紧 | 平台须主动审核否则连带赔偿 |
| 法定许可 | 转载已发表作品需支付报酬 | 洗稿维权有法可依 |

#### 《广告法》重点条款

| 条款 | 内容要点 | 违规风险 |
|------|---------|---------|
| 第28条 | 虚假广告定义（含使用虚假数据） | 网红种草夸大功效属于违法 |
| 第38条 | 麻醉药品/精神药品/医疗器械等禁止做广告 | 医美/护肤品广告限制极严 |
| 第28条 | 「国家级」「最佳」等绝对化用语违法 | 商品测评类内容高风险 |
| 第43条 | 未经用户同意不得发送短信/推送 | 群发推广有法律风险 |

#### 《互联网广告管理办法》2023重点

| 条款 | 内容要点 | 对平台影响 |
|------|---------|----------|
| 第9条 | 互联网广告须有「广告」标识 | 小红书种草笔记须标注「广告」 |
| 第10条 | 弹出广告须设关闭按钮 | 直播中弹窗广告须合规 |
| 第16条 | 直播营销构成广告须遵守广告法 | KOL直播带货适用广告法 |
| 第20条 | 平台对广告内容有审核义务 | 平台须建立广告审核机制 |

### 4.3 平台自治规范引用的法律依据

各平台《社区公约》在用户协议中通常引用以下法律依据：
- 《网络安全法》第47条（平台连带责任）
- 《治安管理处罚法》第25条（散布谣言可拘留）
- 《刑法》第221条（损害商业信誉、商品声誉罪）
- 《刑法》第246条（诽谤罪）
- 《刑法》第253条之一（侵犯公民个人信息罪）
- 《个人信息保护法》2021.11.1

## 五、合规操作建议

### 5.1 内容发布 Checklist

```
□ 内容不涉及政治敏感人物/事件
□ 不使用未授权的影视/音乐/图片素材
□ 商业广告内容已标注「广告」（尤其是小红书、知乎）
□ 涉及商品功效测评有数据支撑，无绝对化用语
□ 涉及医疗/健康/金融领域已具备相应资质
□ 个人信息不侵犯他人隐私
□ 留言互动不引战、不使用歧视性语言
□ 直播内容不诱导打赏、不涉及未成年人
□ 搬运内容已获授权或有合理使用依据
□ 已购买平台认可的广告推广服务（如DOU+、花火）
```

### 5.2 风险预警等级

| 风险等级 | 典型违规行为 | 后果 |
|---------|------------|------|
| 🔴 极高风险 | 涉政、暴恐、邪教、涉未成年人色情 | 永久封禁+公安处理 |
| 🟠 高风险 | 侵权盗版、虚假广告、医疗非法宣传 | 封禁账号+平台公示 |
| 🟡 中等风险 | 软色情擦边、引战拉踩、刷量 | 限流+警告 |
| 🟢 低风险 | 过度标题党、水内容、隐性营销 | 降低推荐权重 |
